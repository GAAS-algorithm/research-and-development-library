# R&D と純粋数学の関係

## 結論：前工程でも同列でもない — 双方向共創

純粋数学（IAS, IHES 等）は **R&D の前工程** でも **R&D と同列** でもない。  
**双方向にブレイクスルーを引き起こす共創関係** である。

---

## 数学 → R&D のブレイクスルー

| 数学 | R&D への影響 |
|------|-------------|
| 圏論・関手 | 型理論、関数型プログラミング、量子計算の基礎 |
| Langlands 双対 | 数論的暗号、量子場の理論 |
| トポロジー | トポロジカル絶縁体、位相量子計算 |
| 最適化理論 | 機械学習、制御、オペレーションズ |
| 確率論・統計 | ベイズ推論、因果推論、AI |
| 表現論 | 量子力学、対称性の記述 |

---

## R&D → 数学のブレイクスルー

| R&D 領域 | 数学への影響 |
|----------|-------------|
| 弦理論・QFT | ミラー対称性、導来圏、モジュライ空間 |
| 量子計算 | 量子群、表現論、位相不変量 |
| 機械学習 | 最適化理論、統計的学習理論、情報幾何 |
| 暗号 | 数論、楕円曲線、格子 |
| 材料科学 | 結晶学、群論、対称性 |
| 計算機科学 | 証明論、圏論的意味論、型理論 |

---

## スキーマ上の扱い

| 観点 | 設計 |
|------|------|
| **階層** | 前工程／後工程の線形パイプラインにはしない |
| **分類** | `foundational_math` と `frontier_topics` を**同列の軸**として持つ |
| **関連** | エントリは `informs` / `informed_by` で双方向リンク可能 |
| **ブレイクスルー** | 数学起因・R&D 起因の両方を扱う |

```
[純粋数学] ←→ [R&D]
     ↑            ↑
     └── 双方向 ──┘
```

## 実装案：双方向リンク

```json
{
  "id": "entry-math-001",
  "foundational_math": { "framework": "category_theory", "core_concept": "duality" },
  "informs": ["entry-rd-042"],
  "informed_by": []
}

{
  "id": "entry-rd-042",
  "frontier_topics": ["quantum_computing"],
  "informed_by": ["entry-math-001"],
  "informs": []
}
```

`informs` / `informed_by` は任意。双方向の因果・影響関係を記録する。
